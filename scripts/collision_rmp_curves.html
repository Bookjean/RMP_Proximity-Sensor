Collision RMP Distance Curves

Current curves use config/params.yaml collision parameters. The metric plot compares collision_rmp_metric_modulation_radius = 0.40 m with 0.25 m. Here x is clearance distance after subtracting obstacle/control-point radii and margin.

Repulsion Acceleration

repel(x) = repulsion_gain * exp(-x / repulsion_std_dev)

Damping Acceleration While Approaching

damping(x, xd) = -(1 - sigmoid(xd / velocity_gate)) * damping_gain * xd / (x / damping_std_dev + eps)

Effective Collision Acceleration

a_eff = repel + damping, displayed as 0 outside each metric modulation radius because M = 0 there.

Metric Scalar, Clipped at 10000

M(x, xd) = metric_scalar / (x / exploder_std_dev + exploder_eps) * (1 - x / radius)^2 * (1 - sigmoid(xd / velocity_gate))
